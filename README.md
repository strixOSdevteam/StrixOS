# StrixOS 🦉

**Minimal. Fast. Built for Gaming.**

StrixOS is a gaming-focused Linux distribution based on Arch Linux.

## Features
- Linux Zen kernel for low-latency gaming
- Automatic driver detection
- XFCE desktop with Zorin-style taskbar
- Pre-installed gaming stack (Steam, Lutris, Wine, MangoHUD)
- Dark red Strix theme
- Works on low-end hardware and Chromebooks

## Color Palette
- Primary: #CC0000 (Strix Red)
- Background: #0A0A0A (Void Black)
- Accent: #FF1A1A (Neon Red)
